Correpicos es una aplicacion para Android

Cada vez que salgas a correr se registraran diversos 
parametro de la sesion:
- distancia
- duracion
- ritmo
- altitud acumulada
- ruta realizada

Despues podras analizar cada una de las sesiones con parciales cada kilometro, 
tanto en una tabla como en el mapa. Asi como analizar los Km realizados 
cada semana, mes y año.

Dispone de varios parametros configurables:
- distancia/tiempo entre cada punto de registro. Valores bajos permiten mucha 
  precision, valores altos suponen un ahorro de bateria
- Pausa automatica del reloj si te paras.
